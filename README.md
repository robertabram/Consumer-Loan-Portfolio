# Consumer-Loan-Portfolio
The report presents the movements and overall summary of the consumer loan portfolio for individual customers.


# Business Objectives
The report describes the daily summary of portfolio and distribution by branches.
Stakeholders can observe the last 12 months portfolio movement for better comparison.
Some sheets of the report show the summary of consumer loan disbursements and repayments.


# Type of Customers
Physical


# Data Model
Dim Tables : Loan_Dim_By_Date
Fact Tables : Loan_Fact, View_IFRS_Report, Loan_Actions


# Tools Used
Microsoft SQL Server 
Power BI
